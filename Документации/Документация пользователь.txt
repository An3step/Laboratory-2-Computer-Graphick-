Сопроводительная документация для пользователя
1. Описание приложения
Это приложение позволяет вам выбрать директорию, после чего приложение считывает все изображения из директории и выводит в формате таблицы следующую информацию об изображениях:
Название
Расширение
Размер
Разрешение
Глубина цвета
Сжатие

2. Основные функции
Выбор директории: выберите директорию, из которой хотите считать изображения.
Чтение директории: приложение также считает информацию из поддиректорий выбранной директории.
Повторный выбор: нажмите кнопку "Выбор другой папки" и выберите другую папку для сканирования.

3. Поддерживаемые форматы
При сканировании директории выводится информация об изображениях следующих форматов:
-png
-jpg
-gif
-tif
-bmp
-pcx

4. Визуализация
Информация об изображениях выводится в виде таблицы, где каждый столбец соответствует пункту информации об изображении.

5. Советы по использованию
Экспериментируйте с различными директориями и форматами изображений. Приложение позволит вам узнать различную информацию об изображениях.
