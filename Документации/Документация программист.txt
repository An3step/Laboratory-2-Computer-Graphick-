Сборка проекта:
-Необходимо наличие консоли Qt 5.x.x (MinGW 64-bit).
-Шаги сборки:
Запустить консоль Qt MinGW.
Использовать команду cd 'путь к папке с исходным ходом'
В папке с исходным кодом использовать команду windeployqt 'Название файла'.exe
Запустить исполняемый файл

1. Описание проекта
Это приложение позволяет пользователям считывать информацию об изображениях из папок. В качестве информации об изображении приложение предоставляет:
-Размер изображения.
-Сжатие изображения.
-Разрешение изображения.
-Плотность пикселей изображения.
-Глубина цвета изображения.
-Название изображения.
-Расширение изображения.

2.Архитектура приложения
Класс основного окна: MainWindow
Метод, сканирующий директорию: ScanDirectory
Метод, показывающий таблицу с информацией: ShowTable

3.Классы и методы

MainWindow:
MainWindow(QWidget *parent = nullptr);
~MainWindow();
void ScanDirectory(QString);
void ShowTable();
Слоты:
void on_ChoiceFolder_clicked();
void on_ReChoice_clicked();

4. Взаимодействие с интерфейсом
Используйте кнопку "Выбрать директорию" для выбора директории.
Используйте кнопку "Выбор другой директории" для выбора другой директории.
